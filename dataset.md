# 🚦 Road Accident Severity in India (Dataset 2)

## 📂 Dataset Information

- **Dataset Name:** Road Accident Severity in India (Dataset 2)  
- **Source:** Kaggle  
- **Link:** https://www.kaggle.com/datasets/saurabhshahane/road-traffic-accidents  

---

## 📊 Description
This dataset contains information about road accidents, including details about drivers, vehicles, road conditions, weather, and accident severity.

---

## 🔑 Key Features
- Time of accident  
- Day of week  
- Number of vehicles involved  
- Number of casualties  
- Driver age and experience  
- Vehicle type  
- Road surface and condition  
- Weather conditions  
- Cause of accident  
- Accident severity  

---

## 🎯 Usage
This dataset can be used for:
- Exploratory Data Analysis (EDA)  
- Pattern identification  
- Accident severity analysis  
- Machine Learning models
